# Sign-up Form

A modern sign-up page built with **HTML5** and **CSS3** as part of **The Odin Project Foundations** curriculum.

## Preview

![Project Preview](assets/sign-up-form.png)

## Features

- Two-column layout using Flexbox
- Full-height sidebar with background image
- Custom logo section with transparent overlay
- Responsive form layout using CSS Grid
- Input focus and validation styles
- Styled "Create Account" button
- Clean and accessible semantic HTML

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid

## Lessons Practiced

- Semantic HTML
- Form elements and validation
- CSS Flexbox
- CSS Grid
- Custom fonts
- Pseudo-classes (`:focus`, `:user-invalid`)
- Box shadows and styling
